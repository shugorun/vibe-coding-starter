## ステータス

- `📋 todo` - 未着手
- `📝 draft` - 草案
- `🔍 reviewed` - 含む一方通行決定をユーザ確認済
- `🚧 implementing` - 実装中
- `✅ done` - 初版完了
- `superseded` - 後続に置き換え
