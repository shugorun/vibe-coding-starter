### 決めたこと

- Mac 対応はフックの Node 化で実現する。理由: catchup 初期化方式や settings の OS 分岐より手間が最小で、Node は Claude Code 自体が必ず要求するので追加依存にならない。
- secret の deny は `.env.*` 全体ではなく `.env` / `.env.local` / `.env.*.local` に限定する。理由: `.env.example`（`.gitignore` で `!` 除外＝コミット対象のテンプレ）を編集可能に保つため。`.env.production` 等に secret を置く運用なら各プロジェクトで deny を追加する。
- `Read(//**)` は意図（どこでも読める）が効いている可能性があり、誤って狭めるリスクを避けて据え置いた。
- LICENSE は MIT、著作権者は暫定で `shugorun`。
